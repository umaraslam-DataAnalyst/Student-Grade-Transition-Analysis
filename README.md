# Student-Grade-Transition-Analysis
A Power BI dashboard analyzing grade migration and student performance trends from Class Tests to Midterms
I teach Maths. But this term, I analyzed my own students like a data analyst.
As part of my MSBA, I built a full performance analysis system for students across Grades 6, 7 and 8 at The Aster School — using nothing but Excel.
Here's what the data revealed:
Linear Regression — Class Test scores predict Term Exam results with r = 0.749. But they only explain 56% of the variance. The other 44%? That's attendance, habits, and effort.
Transition Matrix — I mapped every student's journey from Class Test grade to Term Exam grade (A/B/C/D/F). Some students who scored B in class tests dropped to F in the term. That's not a teaching problem — that's an exam-pressure problem.
 19 students were flagged Academic Risk or At-Risk before results were even finalised.
 Final Term Prediction — using transition probabilities, I built a model that forecasts where each student will land in their final term.
The most surprising finding? Attendance is not enough. Some of my most present students underperformed badly. Showing up is step one. Engaging is step two.
